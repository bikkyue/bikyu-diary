# 要件整理

## 技術スタック

- cloudflare pages/ workers / D1 を利用する。
- フレームワークはAstroを使用する。
- 画像ストレージ：Cloudflare R2

## 機能

### 日記機能
- 日記を表示、投稿、編集、削除を行う。
- 投稿編集削除ができるのは、管理者である私のみ。
- 日記はマークダウン形式で記載する。
- .mdをファイルを直接アップロードができるようにもする。
- タグによる分類・絞り込み

### 画像管理機能
- 画像のアップロード（ドラッグ＆ドロップ対応、複数一括アップロード可）
- アップロード済み画像一覧表示（最新6件）
- 画像URLのコピー（URL、マークダウン形式）
- 画像削除
- 対応フォーマット：jpg, png, gif, webp

## 認証・認可

- Cloudflare Accessを使用する。
- 管理画面は `/admin/*` 配下に配置し、Cloudflare Accessで保護する。
- 一般閲覧者は認証不要で日記を閲覧できる。

## データ設計

日記テーブル（D1）の構造：

| 項目 | 型 | 説明 |
|------|-----|------|
| `id` | INTEGER | 主キー（自動採番） |
| `title` | TEXT | タイトル |
| `slug` | TEXT | URLスラッグ（ユニーク） |
| `content` | TEXT | 本文（マークダウン） |
| `status` | TEXT | 公開状態（`published` / `draft`） |
| `thumbnail` | TEXT | サムネイル画像URL |
| `created_at` | DATETIME | 作成日時 |
| `updated_at` | DATETIME | 更新日時 |

タグテーブル：

| 項目 | 型 | 説明 |
|------|-----|------|
| `id` | INTEGER | 主キー |
| `name` | TEXT | タグ名 |

日記タグ中間テーブル：

| 項目 | 型 | 説明 |
|------|-----|------|
| `diary_id` | INTEGER | 日記ID（外部キー） |
| `tag_id` | INTEGER | タグID（外部キー） |

画像ストレージ（R2）：
- プレフィックス：`images/`
- ファイル名：タイムスタンプ + 拡張子（例：`images/1770564473125.jpeg`）

## UI/UX

- カラーテーマ：ライトモードのみ
- デザイン：シンプル・ミニマル（余白多め、読みやすさ重視）
- レスポンシブ：PC・モバイル両方に十分対応

## 画像・ファイル

- 保存先：Cloudflare R2（バケット名：diary-images）
- ファイルサイズ制限：なし
- 画像処理：アップロードしたままそのまま保存（リサイズ・変換なし）
- 管理画面から画像のアップロード・削除が可能

## OGP/SEO対応

- OGP対応：あり（条件付き）
  - `og:title`, `og:url`, `og:type`, `og:site_name`
  - `twitter:card`, `twitter:title`
- 未対応：`og:description`, `og:image`

## その他

- 検索機能：なし
- RSS/Atomフィード：なし
- ドメイン：Cloudflare Pagesのデフォルトドメインを使用

## 画面一覧

| パス | 説明 | 認証 |
|------|------|------|
| `/` | 日記一覧 | 不要 |
| `/[id]` | 日記詳細 | 不要 |
| `/tags` | タグ一覧 | 不要 |
| `/tag/[name]` | タグ別日記一覧 | 不要 |
| `/admin` | 管理ダッシュボード | 必要 |
| `/admin/new` | 新規日記作成 | 必要 |
| `/admin/edit/[id]` | 日記編集 | 必要 |
| `/admin/upload` | ファイルインポート | 必要 |
| `/admin/images` | 画像管理 | 必要 |
