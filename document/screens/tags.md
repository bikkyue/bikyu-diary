# タグ一覧ページ

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/tags` |
| ファイル | `src/pages/tags.astro` |
| 認証 | 不要（公開ページ） |

## 概要

全タグと各タグの記事数を表示するページ。

## 画面要素

### ヘッダー
- 共通ヘッダー

### メインコンテンツ
- ページタイトル「タグ一覧」
- タグ一覧（カード内）
  - タグ名（タグ別ページへのリンク）
  - 記事数

### フッター
- 共通フッター

## 表示条件

| 状態 | 表示内容 |
|------|---------|
| タグが0件 | 「タグがありません。」 |
| タグが1件以上 | タグをリスト表示（記事数の降順） |

## データ取得

- `getAllTags()` 関数でタグと記事数を取得
- 公開済み日記に紐づくタグのみ表示
- 記事数0のタグは非表示
