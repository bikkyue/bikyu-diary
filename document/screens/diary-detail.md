# 日記詳細ページ

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/[id]` |
| ファイル | `src/pages/[id].astro` |
| 認証 | 不要（公開ページ） |

## 概要

個別の日記の本文を表示するページ。マークダウンをHTMLに変換して表示。

## パラメータ

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| `id` | number | 日記ID |

## 画面要素

### ヘッダー
- 共通ヘッダー

### メインコンテンツ
- 日記タイトル（h1）
- 投稿日
- タグ一覧（タグ別ページへのリンク）
- サムネイル画像（設定されている場合、[画像機能仕様](../features/images.md)参照）
- 本文（マークダウン → HTML変換）
- 「一覧に戻る」ボタン

### フッター
- 共通フッター

## 表示条件

| 状態 | 動作 |
|------|------|
| 日記が存在しない | トップページへリダイレクト |
| 日記が下書き状態 | トップページへリダイレクト |
| 日記が公開状態 | 本文を表示 |

## マークダウン対応

- 見出し（h1〜h6）
- リスト（ul, ol）
- コードブロック（シンタックスハイライトなし）
- インラインコード
- 引用（blockquote）
- リンク
- 画像（[画像機能仕様](../features/images.md)参照）
- 太字、斜体

## OGP対応

SNSシェア時に適切な情報を表示するため、以下のメタタグを出力：

| メタタグ | 値 |
|---------|-----|
| `og:title` | 日記のタイトル |
| `og:url` | ページの完全URL（例: `https://example.com/123`） |
| `og:type` | `article` |
| `og:site_name` | `Diary` |
| `twitter:card` | `summary` |
| `twitter:title` | 日記のタイトル |
