# トップページ

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/` |
| ファイル | `src/pages/index.astro` |
| 認証 | 不要（公開ページ） |

## 概要

公開済みの日記一覧を新着順で表示するページ。

## 画面要素

### ヘッダー
- サイトタイトル「DiaryPage」（トップページへのリンク）
- ナビゲーション
  - 日記一覧（`/`）
  - タグ（`/tags`）

### メインコンテンツ
- ページタイトル「日記一覧」
- 日記カード（0件以上）
  - タイトル（日記詳細へのリンク）
  - 投稿日
  - タグ（タグ別ページへのリンク）

### フッター
- コピーライト表示

## 表示条件

| 状態 | 表示内容 |
|------|---------|
| 日記が0件 | 「まだ日記がありません。」 |
| 日記が1件以上 | 日記カードを新着順に表示 |
| DB接続エラー | エラーメッセージを表示 |

## データ取得

- `getPublishedDiaries()` 関数で `status = 'published'` の日記を取得
- タグ情報も結合して取得
- 作成日時の降順でソート
