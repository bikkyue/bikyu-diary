# 新規日記作成

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/admin/new` |
| ファイル | `src/pages/admin/new.astro` |
| 認証 | Cloudflare Accessで保護 |

## 概要

新しい日記を作成するフォームページ。

## 画面要素

### ヘッダー
- 共通ヘッダー（管理画面リンク表示）

### メインコンテンツ

#### フォーム

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| タイトル | text | ○ | 日記のタイトル |
| 本文 | textarea | ○ | マークダウン形式 |
| タグ | text | - | カンマ区切りで入力 |
| サムネイルURL | text | - | 画像URL（[画像機能仕様](../features/images.md)参照） |
| 公開状態 | select | ○ | 下書き / 公開 |

#### ボタン
- 「保存」ボタン
- 「キャンセル」ボタン（管理画面へ戻る）

### フッター
- 共通フッター

## 操作

### 保存
1. フォームに入力
2. 「保存」ボタンをクリック
3. `POST /api/diary` を呼び出し
4. 成功時、管理画面へリダイレクト
5. エラー時、アラート表示

## バリデーション

| フィールド | ルール |
|-----------|--------|
| タイトル | 必須 |
| 本文 | 必須 |
| 公開状態 | draft または published |
