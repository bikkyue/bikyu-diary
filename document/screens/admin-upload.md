# ファイルインポート

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/admin/upload` |
| ファイル | `src/pages/admin/upload.astro` |
| 認証 | Cloudflare Accessで保護 |

## 概要

マークダウンファイル（.md）をアップロードして日記を作成するページ。

> **注意:** このページは**mdファイルのインポート**専用です。画像アップロード機能については[画像機能仕様](../features/images.md)を参照してください。

## 画面要素

### ヘッダー
- 共通ヘッダー（管理画面リンク表示）

### メインコンテンツ

#### 説明
- インポートの使い方説明
- フロントマターのサンプル表示

#### フォーム

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| ファイル | file | ○ | .mdファイル |

#### ボタン
- 「インポート」ボタン
- 「キャンセル」ボタン（管理画面へ戻る）

### フッター
- 共通フッター

## フロントマター対応

```yaml
---
title: 日記のタイトル
tags: タグ1, タグ2
status: draft または published
thumbnail: 画像URL
---

本文をここに書きます...
```

| フィールド | 必須 | デフォルト |
|-----------|------|-----------|
| title | - | ファイル名 |
| tags | - | なし |
| status | - | draft |
| thumbnail | - | なし |

## 操作

### インポート
1. .mdファイルを選択
2. 「インポート」ボタンをクリック
3. `POST /api/import` を呼び出し（multipart/form-data）
4. 成功時、成功メッセージを表示
5. エラー時、エラーメッセージを表示

## バリデーション

| チェック | エラーメッセージ |
|----------|----------------|
| ファイル未選択 | 「ファイルが選択されていません」 |
| 拡張子が.md以外 | 「マークダウンファイル(.md)のみ対応」 |
