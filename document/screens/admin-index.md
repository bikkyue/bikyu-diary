# 管理画面トップ

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/admin/` |
| ファイル | `src/pages/admin/index.astro` |
| 認証 | Cloudflare Accessで保護 |

## 概要

全日記の管理一覧。下書きを含む全記事を表示し、編集・削除が可能。

## 画面要素

### ヘッダー
- 共通ヘッダー（管理画面リンク表示）

### メインコンテンツ

#### アクションバー
- ページタイトル「日記管理」
- 「+ 新規作成」ボタン（`/admin/new`へ）
- 「ファイルインポート」ボタン（`/admin/upload`へ）

#### 日記一覧
各日記カードに以下を表示：
- タイトル（日記詳細へのリンク）
- 投稿日
- ステータスバッジ（公開/下書き）
- タグ
- 「編集」ボタン（`/admin/edit/[id]`へ）
- 「削除」ボタン（確認ダイアログ後、API呼び出し）

### フッター
- 共通フッター

## 操作

### 削除
1. 「削除」ボタンをクリック
2. 確認ダイアログ「{タイトル}を削除しますか？」
3. OKで `DELETE /api/diary/[id]` を呼び出し
4. 成功時、ページをリロード

## 表示条件

| 状態 | 表示内容 |
|------|---------|
| 日記が0件 | 「まだ日記がありません。」 |
| 日記が1件以上 | 全日記を新着順に表示 |
