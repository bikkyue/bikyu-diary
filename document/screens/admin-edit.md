# 日記編集

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/admin/edit/[id]` |
| ファイル | `src/pages/admin/edit/[id].astro` |
| 認証 | Cloudflare Accessで保護 |

## 概要

既存の日記を編集するフォームページ。

## パラメータ

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| `id` | number | 日記ID |

## 画面要素

### ヘッダー
- 共通ヘッダー（管理画面リンク表示）

### メインコンテンツ

#### フォーム（既存データで初期化）

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| タイトル | text | ○ | 日記のタイトル |
| 本文 | textarea | ○ | マークダウン形式 |
| タグ | text | - | カンマ区切りで入力 |
| サムネイルURL | text | - | 画像URL（[画像機能仕様](../features/images.md)参照） |
| 公開状態 | select | ○ | 下書き / 公開 |

#### ボタン
- 「更新」ボタン
- 「キャンセル」ボタン（管理画面へ戻る）

### フッター
- 共通フッター

## 操作

### 更新
1. フォームを編集
2. 「更新」ボタンをクリック
3. `PUT /api/diary/[id]` を呼び出し
4. 成功時、管理画面へリダイレクト
5. エラー時、アラート表示

## 表示条件

| 状態 | 動作 |
|------|------|
| 日記が存在しない | 管理画面へリダイレクト |
| 日記が存在する | フォームに既存データを表示 |
